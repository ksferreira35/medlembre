# Checklist da Entrega Final

## Banco de Dados

- Criar projeto no Supabase.
- Executar `supabase/schema.sql` no SQL Editor.
- Configurar `MEDLEMBRE_STORAGE=supabase`.
- Configurar `SUPABASE_URL`.
- Configurar `SUPABASE_ANON_KEY`.
- Rodar a aplicação e testar cadastro, listagem, marcar como tomado e remoção.

## GitHub

- Criar uma issue para cada tarefa.
- Criar uma branch para cada aluno.
- Cada aluno deve abrir pelo menos 1 Pull Request.
- Cada PR deve ser revisado e aprovado por outro integrante.
- Garantir que o GitHub Actions fique verde antes do merge.

## Deploy

- Atualizar a imagem/serviço publicado com a versão final.
- Configurar as variáveis do Supabase no ambiente de deploy.
- Testar a aplicação publicada usando o banco em nuvem.

## README e PDF

- Preencher nome completo e matrícula de todos os integrantes.
- Conferir link do repositório público.
- Conferir link do deploy.
- Citar Supabase como banco de dados em nuvem.
- Enviar o mesmo PDF para todos os integrantes.
